# Blog App.

I will add documentation for API endpoints using the gem rswag


## Built With:

- Ruby
- Postgresql

To get a local copy up and running follow these simple steps.

## Getting Started


### Prerequisites
- Ruby
- Postgresql
- Rails

## Setup

### Clone this repository
git clone https://github.com/looh16/blog_app


### Run project
- cd blog_app
- run rails server
